## Placeholder Tokens

Use these in scripts to auto-fill info:

| Token | Replaces With |
|-------|---------------|

| `[LAST]` | Last name |
| `[FULL_NAME]` | Full name |
| `[DOB]` | Date of birth |
| `[PHONE]` | Phone number |
| `[ADDRESS]` | Full address |
| `[INSURANCE]` | Insurance ID |
| `[PHARMACY]` | Pharmacy name |
